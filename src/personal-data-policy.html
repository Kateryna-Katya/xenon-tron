Data Privacy & Security

Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки данных, предпринимаемые проектом XenonTron (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этой информации в рамках современной цифровой инфраструктуры.

1.1. Важнейшей целью Оператора является соблюдение прав и свобод человека при обработке его данных, включая защиту права на неприкосновенность частной жизни и личную тайну.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях (далее — «Пользователи») веб-сайта xenon-tron.blog.

2. Основные понятия

  • Веб-сайт — совокупность графических материалов, программ для ЭВМ и баз данных платформы XenonTron.
  • Пользователь — любой посетитель веб-сайта, интересующийся практиками применения ИИ.
  • Персональные данные — любая информация, относящаяся к определенному Пользователю.
  • Обработка данных — любое действие (сбор, запись, систематизация, хранение, использование), совершаемое с помощью средств автоматизации или без них.

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Данные, предоставляемые Пользователем:
    • Имя (для персонализации обращений);
    • Адрес электронной почты (для связи и рассылок);
    • Номер телефона (для экспертной поддержки).
  • Данные, собираемые автоматически:
    • Обезличенные данные о посетителях (файлы cookie, IP-адрес, тип браузера) с помощью сервисов Google Analytics и аналогичных инструментов для улучшения работы AI-интерфейсов.

4. Цели обработки

  • Идентификация Пользователя для предоставления доступа к образовательным материалам.
  • Установление обратной связи, включая обработку запросов на внедрение AI-решений.
  • Информирование о новых стратегиях роста и инновационных технологиях (с согласия Пользователя).
  • Аналитика использования платформы для улучшения пользовательского опыта.

5. Правовые основания

Оператор обрабатывает данные только при наличии:

  • Добровольного согласия Пользователя, выраженного через заполнение форм на сайте.
  • Разрешения в настройках браузера Пользователя (использование cookie).

6. Безопасность и сроки хранения

  • Оператор принимает все технические меры для защиты данных от неправомерного доступа, используя передовые протоколы безопасности.
  • Персональные данные никогда не передаются третьим лицам, за исключением случаев исполнения законодательства ЕС или наличия явного согласия Пользователя.
  • Срок обработки данных является неограниченным, однако Пользователь может в любой момент отозвать согласие, написав нам на почту.

7. Заключительные положения

Пользователь может получить любые разъяснения по вопросам обработки данных, обратившись по адресу: hello@xenon-tron.blog.

Актуальная версия Политики находится в свободном доступе на сайте xenon-tron.blog. В документе будут отражаться любые изменения, вызванные развитием технологий или законодательства.
